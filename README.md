# NYC-Taxi-EDA-Project
## Abstract
Exploratory Data Analysis on The Kaggle NYC Taxi Trip Duration Dataset to build a model that predicts the total ride duration of taxi trips in New York City.

## Dataset
https://www.kaggle.com/c/nyc-taxi-trip-duration/data
